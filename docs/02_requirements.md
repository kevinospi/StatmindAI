# Requerimientos Funcionales
    Aprender Estadística
        El usuario podrá seleccionar preguntas predefinidas sobre conceptos básicos de estadística y probabilidad.
        El usuario podrá realizar preguntas personalizadas relacionadas con estadística y probabilidad.
        El sistema deberá generar explicaciones en lenguaje natural orientadas al aprendizaje.
        El sistema podrá mostrar fórmulas matemáticas cuando sean necesarias para la explicación.
        El sistema deberá proporcionar ejemplos prácticos para facilitar la comprensión de los conceptos.
    Analizar Datos
        El usuario podrá cargar archivos en formato CSV.
        El usuario podrá cargar archivos en formato Excel.
        El sistema deberá validar que el archivo tenga un formato válido.
        El sistema deberá generar automáticamente estadísticas descriptivas básicas.
        El sistema deberá generar visualizaciones básicas de los datos.
        El usuario podrá solicitar análisis estadísticos específicos mediante lenguaje natural.
        El sistema deberá interpretar los resultados obtenidos y explicarlos en lenguaje natural.
# Requerimientos No Funcionales
    El backend deberá desarrollarse utilizando FastAPI.
    La aplicación deberá exponer una API REST.
    El proyecto deberá utilizar control de versiones mediante Git y GitHub.
    El sistema deberá ejecutarse en contenedores Docker.
    La arquitectura deberá ser modular para facilitar futuras ampliaciones.
    El sistema deberá separar los cálculos estadísticos de las respuestas generadas por IA.
    Las respuestas generadas deberán ser comprensibles para usuarios con conocimientos básicos de estadística.
